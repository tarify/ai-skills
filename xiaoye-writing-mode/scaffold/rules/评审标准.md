# 评审标准

## 总体评分体系

采用五星制评分，总分100分转换为星级：

| 分数区间 | 星级 | 评价 |
|---------|------|------|
| 90-100分 | ⭐⭐⭐⭐⭐ | 优秀 |
| 75-89分 | ⭐⭐⭐⭐ | 良好 |
| 60-74分 | ⭐⭐⭐ | 合格 |
| 40-59分 | ⭐⭐ | 需改进 |
| 0-39分 | ⭐ | 不合格 |

## 评分维度与权重

### 一、逻辑一致性（权重40%，共40分）

#### 1.1 因果链条（10分）

| 等级 | 标准 | 分值 |
|------|------|------|
| 优秀 | 事件发展因果清晰，转折自然，无逻辑跳跃 | 9-10分 |
| 良好 | 主要因果链条完整，有个别小转折稍显生硬 | 7-8分 |
| 合格 | 基本逻辑通顺，但存在可识别的因果关系缺失 | 5-6分 |
| 需改进 | 因果链条有明显断裂，读者需要自行脑补 | 3-4分 |
| 不合格 | 逻辑混乱，事件发展缺乏合理性 | 0-2分 |

**检查要点：**
- [ ] 每个事件是否有明确的前因后果
- [ ] 转折是否铺垫充分
- [ ] 时间线是否连贯
- [ ] 有无突兀的情节跳跃

#### 1.2 人物动机（10分）

| 等级 | 标准 | 分值 |
|------|------|------|
| 优秀 | 人物行为有充分动机支撑，符合性格设定，无OOC | 9-10分 |
| 良好 | 主要行为动机合理，个别次要行为动机略显薄弱 | 7-8分 |
| 合格 | 核心行为有动机，但部分行为缺乏解释 | 5-6分 |
| 需改进 | 人物行为经常缺乏动机，性格表现不稳定 | 3-4分 |
| 不合格 | 人物行为完全不符合设定，严重OOC | 0-2分 |

**检查要点：**
- [ ] 主角重大决策是否有合理动机
- [ ] 次要角色行为是否符合设定
- [ ] 情绪变化是否有铺垫
- [ ] 人物成长是否合理

#### 1.3 世界观遵守（10分）

| 等级 | 标准 | 分值 |
|------|------|------|
| 优秀 | 完全遵守已建立的世界观规则，无任何冲突 | 9-10分 |
| 良好 | 基本遵守世界观，有轻微不一致但可合理解释 | 7-8分 |
| 合格 | 有明显冲突但可修复，需要设定补充说明 | 5-6分 |
| 需改进 | 多处违反世界观设定，需要大幅修改 | 3-4分 |
| 不合格 | 严重违反核心设定，世界观崩溃 | 0-2分 |

**检查要点：**
- [ ] 力量体系使用是否合理
- [ ] 地理设定是否一致
- [ ] 时间线是否吻合
- [ ] 组织/势力设定是否遵守

#### 1.4 角色认知边界（10分）

> 角色只能说出他知道的信息，不能开上帝视角

| 等级 | 标准 | 分值 |
|------|------|------|
| 优秀 | 所有角色严格遵循认知边界，无任何越界 | 9-10分 |
| 良好 | 基本遵循，有个别轻微越界但不影响阅读 | 7-8分 |
| 合格 | 有明显越界但可修复 | 5-6分 |
| 需改进 | 多处越界，角色说出不该知道的信息 | 3-4分 |
| 不合格 | 严重越界，角色频繁使用不应掌握的术语或信息 | 0-2分 |

**检查要点：**
- [ ] 角色是否使用了不应掌握的专业术语（如首次遭遇的敌人，不应直接叫出其分类名）
- [ ] 角色是否说出了他无法获知的信息
- [ ] 角色是否表现出与其经历不符的认知水平
- [ ] 对话中是否有"为学生讲解"式的信息倾倒

**典型违规：**
- 首次遭遇敌人，角色直接叫出其完整分类名称
- 新兵问出老兵才应该知道的基础问题
- 角色在对话中百科式讲解设定信息

---

### 一、逻辑一致性（权重40%，共40分）

> 因果链条10分 + 人物动机10分 + 世界观遵守10分 + 角色认知边界10分

---

### 二、可读性评估（权重30%，共30分）

#### 2.1 语言流畅度（5分）

| 等级 | 标准 | 分值 |
|------|------|------|
| 优秀 | 句式自然优美，长短句搭配得当，阅读流畅 | 9-10分 |
| 良好 | 语言流畅，偶有冗长或生硬句式 | 7-8分 |
| 合格 | 基本可读，有明显可优化空间 | 5-6分 |
| 需改进 | 语言生硬，句式单调，阅读体验差 | 3-4分 |
| 不合格 | 语句不通，严重影响阅读 | 0-2分 |

**检查要点：**
- [ ] 长短句是否搭配合理
- [ ] 是否有冗长累赘的表达
- [ ] 用词是否准确得当
- [ ] 是否符合中文阅读习惯

#### 2.2 设定融入叙事（10分）

> 设定信息应通过对话和感官描写自然展现，不独立成段，不使用百科式讲解

| 等级 | 标准 | 分值 |
|------|------|------|
| 优秀 | 所有设定信息自然融入叙事，读者无"出戏"感 | 9-10分 |
| 良好 | 大部分设定融入自然，个别处略显生硬 | 7-8分 |
| 合格 | 有明显设定说明段落，但不严重影响阅读 | 5-6分 |
| 需改进 | 大量生硬设定说明，频繁"出戏" | 3-4分 |
| 不合格 | 设定与叙事完全割裂，像在读百科词条 | 0-2分 |

**检查要点：**
- [ ] 设定信息是否通过对话自然展现（而非角色讲解）
- [ ] 是否有独立成段的设定说明
- [ ] 数据/参数是否用感官描写替代（"100索精神力"→"跨过四等的门槛"）
- [ ] 伤亡/战力等数据是否用感受替代百分比

**典型违规：**
- 角色在对话中报出精确数据（"伤亡30%"、"半径500公里"）
- 内心独白罗列理论术语（"人民战争、统一战线、持久战、矛盾论、实践论"）
- 角色之间百科式讲解（"这个种族有三条进化路线……"）

#### 2.3 对话合理性（10分）

> 对话应该是平等的战士交流，不应问已知/显而易见的问题

| 等级 | 标准 | 分值 |
|------|------|------|
| 优秀 | 所有对话自然合理，符合角色身份和认知水平 | 9-10分 |
| 良好 | 对话基本合理，个别处略显刻意 | 7-8分 |
| 合格 | 有明显不合理的对话，但可修复 | 5-6分 |
| 需改进 | 多处对话不合理，角色显得无知或刻意 | 3-4分 |
| 不合格 | 对话严重不合理，角色像在互相讲解 | 0-2分 |

**检查要点：**
- [ ] 角色是否问了他已经知道的问题
- [ ] 角色是否问了显而易见的问题
- [ ] 对话是否是平等交流（而非学生向老师提问）
- [ ] 是否有"为读者讲解"式的对话

**典型违规：**
- 守了两个月的战士问"这个地方是哪里？"
- 刚经历完撤退的人问"我们打不过它？"
- 角色问出答案显而易见的问题，只为给作者一个讲解的机会

#### 2.4 情感传达（5分）

| 等级 | 标准 | 分值 |
|------|------|------|
| 优秀 | 情感真挚自然，能引发读者共鸣 | 9-10分 |
| 良好 | 情感表达清晰，有共鸣点 | 7-8分 |
| 合格 | 情感表达基本到位，但略显平淡 | 5-6分 |
| 需改进 | 情感表达生硬或过度煽情 | 3-4分 |
| 不合格 | 情感表达失败，无法引起共鸣 | 0-2分 |

**检查要点：**
- [ ] 情绪转折是否自然
- [ ] 是否有过度的煽情
- [ ] 情感细节是否充分
- [ ] 是否避免了形容词堆砌

---

### 三、创新性（权重20%，共20分）

#### 3.1 情节设计（10分）

| 等级 | 标准 | 分值 |
|------|------|------|
| 优秀 | 情节新颖独特，有意外惊喜，不落俗套 | 9-10分 |
| 良好 | 有创新元素，部分情节有亮点 | 7-8分 |
| 合格 | 情节合理但较常规，无明显创新 | 5-6分 |
| 需改进 | 情节老套，缺乏新意 | 3-4分 |
| 不合格 | 情节陈旧，完全复制常见套路 | 0-2分 |

**检查要点：**
- [ ] 是否有常见套路的创新运用
- [ ] 转折是否有惊喜感
- [ ] 是否有独特的情节设计
- [ ] 是否避免了过度俗套

#### 3.2 人物塑造（10分）

| 等级 | 标准 | 分值 |
|------|------|------|
| 优秀 | 人物立体饱满，有独特记忆点，印象深刻 | 9-10分 |
| 良好 | 人物有特色，部分角色较为鲜明 | 7-8分 |
| 合格 | 人物基本完整，但缺乏独特性 | 5-6分 |
| 需改进 | 人物扁平，缺乏个性 | 3-4分 |
| 不合格 | 人物塑造失败，毫无记忆点 | 0-2分 |

**检查要点：**
- [ ] 主角是否有独特特征
- [ ] 配角是否有差异化
- [ ] 人物关系是否丰富
- [ ] 是否有记忆点

---

### 四、完整性（权重10%，共10分）

#### 4.1 内容完整（5分）

| 等级 | 标准 | 分值 |
|------|------|------|
| 优秀 | 内容完整，无遗漏，结构严谨 | 9-10分 |
| 良好 | 基本完整，有轻微可补充之处 | 7-8分 |
| 合格 | 有明显遗漏但不影响整体 | 5-6分 |
| 需改进 | 内容不完整，影响阅读体验 | 3-4分 |
| 不合格 | 严重缺失，无法构成完整章节 | 0-2分 |

**检查要点：**
- [ ] 章节是否有完整的开头、发展、结尾
- [ ] 关键情节是否充分展开
- [ ] 是否有未交代的悬念（刻意留白除外）
- [ ] 场景切换是否完整

#### 4.2 钩子设置（5分）

| 等级 | 标准 | 分值 |
|------|------|------|
| 优秀 | 悬念设置巧妙，强烈吸引读者继续阅读 | 9-10分 |
| 良好 | 有有效钩子，能激发阅读兴趣 | 7-8分 |
| 合格 | 有悬念设置，但吸引力一般 | 5-6分 |
| 需改进 | 悬念设置生硬或无效 | 3-4分 |
| 不合格 | 无悬念，读者无继续阅读欲望 | 0-2分 |

**检查要点：**
- [ ] 章末是否有悬念或期待点
- [ ] 钩子是否自然不生硬
- [ ] 是否有吸引读者的元素
- [ ] 是否设置了下章期待

## 问题分级标准

### 🔴 严重问题（必须修改）
- 逻辑严重断裂
- 人物严重OOC
- 世界观崩溃
- 角色认知边界严重越界（角色说出完全不该知道的信息）
- 大量生硬设定说明，严重破坏阅读体验
- 严重影响阅读的问题

### 🟡 中等问题（建议修改）
- 明显的逻辑漏洞
- 人物行为动机不足
- 设定冲突
- 角色问出已知/显而易见的问题
- 设定信息生硬融入，有明显"出戏"感
- 节奏明显失衡

### 🟢 轻微问题（可选优化）
- 表达不够精准
- 细节描写可增强
- 情感表达可加强
- 局部节奏调整
- 个别设定融入略显生硬

---

## 评审流程

1. **快速浏览**：整体感受，初步判断
2. **逐段分析**：应用评分标准逐项打分
3. **问题识别**：按严重程度分类
4. **建议生成**：提供具体改进方案
5. **综合评估**：给出总评和星级

---

## 注意事项

1. **客观性**：基于标准评分，避免主观偏好
2. **建设性**：指出问题的同时提供解决方案
3. **一致性**：同一标准贯穿始终
4. **尊重创作意图**：在理解作者意图基础上评估
5. **角色认知边界**：角色只能说出他知道的信息，不能开上帝视角
6. **设定融入叙事**：设定信息应自然展现，不应独立成段或百科式讲解
7. **对话合理性**：对话应是平等的战士交流，不应为讲解而问
8. **战士优先原则**：功能兵种首先是战士，不应贬低其战士身份
